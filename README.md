# React Portfolio
https://still-badlands-62849.herokuapp.com/



## Description:
A portfolio website featuring my info about me, highlighted project, my resume and an interactive contact form.

## Installation:
Download node.js

To create a react app:
 > npx react-portfolio 

Download all dependencies 
 > npm i  

Start the server:
  > npm start
